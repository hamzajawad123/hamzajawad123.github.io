Bellabeat Case Study

Google Data Analytics Capstone Project using R

Introduction

This Bellabeat case study analyzes smart device usage data to help Bellabeat, a wellness technology company, understand customer behavior. The goal is to derive actionable insights for marketing and product development.

Objective

  • Explore Fitbit Fitness Tracker data to understand user activity, sleep, and wellness patterns.
  • Perform data cleaning, analysis, and visualization using R.
  • Generate insights to guide Bellabeatâ€™s product and marketing strategies.

Data Cleaning and Preparation

  1. Imported datasets using read_csv().
  2. Standardized column names using clean_names() from the janitor package.
  3. Converted date/time columns using ymd() or mdy_hms().
  4. Removed duplicates, missing values, and unrealistic entries (e.g., zero sleep minutes).
  5. Created derived columns like total_daily_steps and average_sleep_hours.
  6. Aggregated data to summarize daily and weekly activity for trend identification.

Analysis & Key Findings

  • Average daily steps were below 10,000, showing potential for increased activity.
  • Sleep duration was higher on weekends than weekdays.
  • Sedentary minutes negatively correlated with active minutes.
  • Heart rate and weight showed gradual consistency with activity levels.

Conclusion

The analysis uncovered actionable insights that can help Bellabeat improve user engagement and product personalization. Personalized wellness programs and targeted marketing could increase overall user satisfaction and encourage healthier habits.

Recommendations

  • Targeted challenges to promote weekday activity increases.
  • Personalized notifications to remind users to move after long sedentary periods.
  • Marketing emphasis on improved sleep tracking and wellness awareness.
  • Product features focusing on weekend activity monitoring.

Summary

  • Tools Used R, tidyverse, dplyr, ggplot2
  • Data Fitbit Fitness Tracker Dataset
  • Skills Data Cleaning, EDA, Visualization, Insight Generation
