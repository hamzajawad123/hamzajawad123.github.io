Cyclistic Ride Patterns Analysis

Google Data Analytics Capstone Project using R

Introduction

This capstone project focuses on analyzing Cyclistic bike-share data from the first quarter of 2019 and 2020. The goal is to understand how annual members and casual riders use Cyclistic bikes differently, helping the company design strategies to convert casual riders into annual members.

Objective

  • Clean and merge raw Divvy trip datasets for 2019 Q1 and 2020 Q1.
  • Analyze and visualize ride duration, user type, day, and seasonal patterns using R.
  • Generate insights that can support Cyclisticâ€™s marketing strategies.

Data Cleaning and Preparation

  1. Loaded Divvy_Trips_2019_Q1 and Divvy_Trips_2020_Q1 datasets using read_csv().
  2. Renamed and standardized column names with clean_names().
  3. Converted date columns to datetime objects using ymd_hms().
  4. Removed null values, duplicates, and invalid ride durations (<1 min or >24 hours).
  5. Created new columns: ride_length_min, day_of_week, month, year, and season.
  6. Re-coded user type from usertype (subscriber/customer) to member_casual (member/casual).
  7. Merged the 2019 and 2020 datasets into one unified DataFrame for analysis.

Analysis & Findings

  • Members consistently accounted for more rides than casual users.
  • Casual riders showed a sharp increase on weekends, while members were more active on weekdays.
  • Average ride length for casual riders was significantly longer than for members.
  • Ride activity peaked in summer afternoons for both groups.
  • Top 10 starting stations were similar for both groups, with downtown areas being the most popular.

Recommendations

  • Seasonal Campaigns: Launch targeted membership drives ahead of spring/summer peaks.
  • Geo-Targeting: Focus on high-casual stations near leisure/tourist areas.
  • Value Proposition: Highlight cost savings of membership for long rides.
  • Flexible Plans: Introduce weekend/seasonal membership tiers to attract casual riders.

Conclusion

The analysis highlights key behavioral differences between member and casual riders. Casual users prefer long weekend rides, while members use the service more frequently for daily commuting. Cyclistic should target casual riders with weekend-based promotions or discounted memberships to encourage conversion.

Summary

  • Tools Used R, tidyverse, dplyr, ggplot2
  • Data Divvy Trips 2019, 2020 Q1
  • Skills Data Cleaning, EDA, Visualization, Insights
